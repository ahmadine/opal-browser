Dynamic Opal-Browser integration with Roda via Roda-Sprockets
=============================================================

To install, run:

    $ bundle install

Development mode
----------------

    $ bundle exec rackup

Production mode
---------------

    $ bundle exec rake assets:precompile
    $ RACK_ENV=production bundle exec rackup
